#### nginx的常用命令:
```js
$ start nginx                   // 开启nginx
$ nginx -s stop                 // 快速关机
$ nginx -s quit                 // 优雅关机
$ nginx -s reload               // 重新加载
$ nginx -s reopen               // 重新打开日志文件

$ tasklist /fi "imagename eq nginx.exe" // 产看开启的 nginx.exe 列表

```
